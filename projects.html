It's A Bird

This colorful web app created with react and redux is as colorful as a cockatoo. Users can learn about different types of cockatoos and even take a quiz to find out which cockatoo they are most like. The use of flexbox design and responsive images ensures each cockatoo looks fantastic on every screen size.

Visit It's A Bird Web App

reddit Board

The redditBoard web app uses react, redux, and the reddit JSON API to make a fun chalkboard themed site. Users can view posts from a list of subreddits, see comments for posts, and search for content.

Visit redditBoard Web App

Python Terminal Game

This Grand Theft Auto themed text-based terminal game utilizes python classes to let players create their own character. Then they challenge a built-in character to a race or instead rob a bank. They must be savvy with their choices, because if they run out of money, it's GAME OVER...Sadge.
